# Contributing

Thanks for checking out 大卡记.

## Development

Run locally:

```bash
python3 -m http.server 4173
```

Open:

```text
http://127.0.0.1:4173/
```

## Guidelines

- Keep the app dependency-free unless there is a strong reason
- Keep the UI mobile-first
- Do not commit API keys or personal sync URLs
- Prefer small, focused changes
- Test date navigation and food entry on a narrow mobile viewport

## Commit Style

Use concise imperative commit messages, for example:

```text
Add food estimate clear button
Fix local date navigation
```
